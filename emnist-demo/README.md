# EMNIST Handwritten Character Recognition Demo

Веб-приложение для распознавания рукописных символов с использованием модели improved_cnn_v5 через ONNX Runtime Web.

## О проекте

- **Модель:** improved_cnn_v5 (SE-CNN), 712K параметров
- **Точность:** 91.15% на EMNIST Balanced
- **Классы:** 47 (10 цифр + 37 букв с уникальным написанием)
- **Инференс:** Полностью в браузере, без сервера
- **Развёртывание:** GitHub Pages (статический хостинг)

## Локальный запуск

Приложение требует HTTP-сервера для загрузки ONNX-модели (file:// не поддерживается).

### Быстрый старт

```bash
# В директории с файлами приложения:
python -m http.server 8000

# Или с Node.js:
npx serve .

# Откройте в браузере: http://localhost:8000
